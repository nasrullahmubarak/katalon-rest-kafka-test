# Katalon REST & Kafka Test

Project ini berisi automation test untuk:
- REST API Student (POST & GET)
- Kafka Consumer (custom keyword)

## Struktur
- Object Repository: definisi request API
- Test Cases: POST_Student_Test, GET_Student_Test, Kafka_Consumer_Test
- Test Suites: API_TestSuite, Kafka_TestSuite
- Reports: hasil eksekusi

## Cara Jalankan
1. Buka Katalon Studio
2. Run Test Suite `API_TestSuite` atau `Kafka_TestSuite`
3. Lihat hasil di folder Reports
